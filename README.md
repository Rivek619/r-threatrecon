# R-Threat Recon Intelligence

A simple OSINT recon dashboard for security researchers and bug bounty hunters.

Built by RivuDon (Rivek Raj Tamang).

---

## What is this?

R-Threat Recon Intelligence is a browser-based tool that helps you quickly generate recon links for any domain.

It groups tools under clear sections like:

- Search Engines  
- Infrastructure & DNS  
- Exposure & Leaks  
- Threat Intelligence  

No backend. No tracking. Just fast recon.

## 📦 How to Use

1. Download or clone the repo  
2. Open `index.html` in your browser  
3. Enter a domain (example: target.com)  
4. Click **Run Recon**  

You can enter multiple domains separated by commas.
